ML Mini-Experiment Notes---

Student Score Prediction using Linear Regression
1. What is this Task All About?

This task is a Machine Learning Mini-Experiment where we build a model that predicts a student's exam score based on the number of hours they study.

Problem Statement
Input  → Study Hours
Output → Predicted Score

Example:

Study Hours	Score
2	25
5	55
8	85

If a student studies for 7 hours, the model predicts the expected score.

3. Why Do We Use Machine Learning?

Traditional programming works like:

Input + Rules = Output

Example:

if hours == 5:
    score = 55

This only works for predefined conditions.

Machine Learning works like:

Data + Output = Learning

The model automatically learns patterns from data and can predict future results.

Benefits
Learns from data
Improves prediction accuracy
Handles large datasets
Reduces manual work
Useful in AI systems

6. Basic ML Terms and Terminology
Machine Learning (ML)

Machine Learning is a branch of Artificial Intelligence that enables computers to learn from data and make predictions.

Dataset

A collection of data used for training.

Example
Hours	Score
2	25
5	55
Feature (X)

Input variable.

Example
Hours Studied

In our project:

X = data[["Hours"]]
Label (Y)

Output variable.

Example
Score

In our project:

y = data["Score"]
Model

A mathematical algorithm that learns patterns from data.

Example
model = LinearRegression()
Training

The process of teaching the model using data.

Example
model.fit(X, y)

Meaning:

Hours → Score relationship learned
Prediction

Using the trained model to estimate future values.

Example
model.predict([[7]])

Output:

75

Meaning:

7 Study Hours → 75 Marks
Linear Regression

A Machine Learning algorithm used to predict continuous values.

Formula
y=mx+b
m
b

Where:

y = Predicted Score
x = Study Hours
m = Slope
b = Intercept
Accuracy

Measures how well the model predicts outcomes.

Higher accuracy means better predictions.

Data Visualization

Representing data using graphs.

Example:

plt.scatter()

Benefits:

Understand trends
Detect patterns
Analyze relationships
Algorithm

A set of mathematical rules used for learning.

Examples:

Linear Regression
Decision Tree
Random Forest
KNN
SVM

In this project:
