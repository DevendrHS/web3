# Staking Smart Contract

## Overview

A Solidity staking smart contract that allows users to stake SCAI coins and earn rewards.

## Features

- Stake Coins
- Reward Calculation
- Withdraw Stake
- Contract Balance Tracking

## Technologies

- Solidity
- Remix IDE
- MetaMask
- SCAI Mainnet

## Deployment

Network: SCAI Mainnet

## Functions

- stake()
- calculateReward()
- withdraw()
- getContractBalance()

## Learning Outcome

Learned staking mechanisms, smart contract deployment, and blockchain reward systems.
