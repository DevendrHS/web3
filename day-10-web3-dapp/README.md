# Internship Reward DApp

## Overview

Internship Reward DApp is a full-stack Web3 decentralized application built using React.js, Node.js, Express.js, MongoDB, Solidity, and MetaMask integration. The application allows interns to connect their wallet, manage internship tasks, and receive ERC20 reward tokens.

---

# Features

- MetaMask Wallet Connection
- ERC20 Reward Token Integration
- React Frontend Dashboard
- Node.js & Express Backend APIs
- MongoDB Database Integration
- Intern Registration System
- Reward Distribution System
- Task Tracking System
- Token Balance Checking
- Full Stack Web3 Architecture
