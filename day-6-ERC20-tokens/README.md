# ERC20 Token Projects

## Objective

Create and deploy multiple ERC20 smart contracts with different real-world use cases using Solidity and OpenZeppelin.

---

# Contracts Included

## 1. Intern Reward Token (IRT)

A reward-based ERC20 token for internship bonuses and achievements.

### Features
- Owner-controlled rewards
- Token minting
- Internship reward system

### Concepts Learned
- ERC20 Standard
- Ownable Contract
- Minting Logic
- Solidity Inheritance

---

## 2. Task Completion Token (TCT)

An ERC20 token that automatically rewards users after task completion.

### Features
- Automatic task rewards
- Task tracking
- Mapping usage

### Concepts Learned
- Solidity Mapping
- Reward Automation
- Dynamic State Management

---

## 3. Attendance Token (ATK)

An ERC20 token that rewards users for daily attendance.

### Features
- Attendance tracking
- Daily rewards
- Timestamp validation

### Concepts Learned
- block.timestamp
- Time-based conditions
- Daily reward systems

---
