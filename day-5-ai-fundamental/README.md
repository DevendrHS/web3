# What is Artificial Intelligence?

Artificial Intelligence (AI) is a technology that enables machines to simulate human intelligence, learn from data, solve problems, and make decisions.

Examples:
- ChatGPT
- Google Assistant
- Recommendation Systems
- Self-driving Cars
- AI Image Generators

---

# Machine Learning (ML)

Machine Learning is a subset of AI where systems learn patterns from data automatically without explicit programming.

## Types of Machine Learning

### Supervised Learning
Learning using labeled datasets.

### Unsupervised Learning
Finding patterns in unlabeled data.

### Reinforcement Learning
Learning using rewards and punishments.

---

# Deep Learning

Deep Learning is a subset of Machine Learning that uses Neural Networks with multiple layers to process complex data such as images, videos, and language.

---

# Neural Networks

Neural Networks are AI models inspired by the human brain.

Structure:
- Input Layer
- Hidden Layers
- Output Layer

# Natural Language Processing (NLP)
