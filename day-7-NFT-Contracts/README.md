# Day 7 — ERC721 NFT Contract

## Objective

Learn and implement ERC721 NFT smart contracts using Solidity and OpenZeppelin.

This module focuses on:
- NFT minting
- certificate NFTs
- achievement badge NFTs
- metadata storage
- ownership management

---

# Contracts Included

## 1. Internship Certificate NFT

An ERC721 NFT contract for issuing blockchain-based internship certificates.

### Features
- NFT certificate minting
- Unique token IDs
- Ownership control

### Concepts Learned
- ERC721 Standard
- NFT Minting
- Token ID Management
- OpenZeppelin ERC721

---

## 2. Course Completion NFT

An ERC721 NFT contract for issuing course completion certificates.

### Features
- Course certificate minting
- Course name storage
- NFT ownership

### Concepts Learned
- Solidity Mapping
- NFT Metadata Storage
- ERC721 Development

---

## 3. Achievement Badge NFT

An ERC721 NFT contract for achievement and milestone badges.

### Features
- Achievement badge minting
- Badge metadata storage
- NFT ownership system

### Concepts Learned
- NFT Badge System
- Metadata Management
- ERC721 Token Logic

---

## 4. NFT Minting Contract

A general ERC721 NFT minting smart contract with metadata URI storage.

### Features
- NFT minting
- tokenURI storage
- Ownership control
- Metadata management

### Concepts Learned
- ERC721URIStorage
- tokenURI
- Metadata URI
- NFT Ownership
